  • jquery/jquery-ui.min.css - jQuery UI custom stylesheets
  • bootstrap/bootstrap.min.css - bootstrap stylesheets
  • fontawesome/font-awesome.min.css - font awesome icons stylesheets
  • summernote/summernote.css - Summernote WYSIWYG editor stylesheets
  • codemirror/codemirror.css - Codemirror stylesheets

  • rickshaw/rickshaw.css - Rickshaw charts stylesheets

In less/css folder of template you will find 5 custom themes for template. Each of this template can be used via including other theme variables.

Please use your own css file for your stylesheets. It's easiest way to update template in future.

How to use builder?

First of all you should to know that this feature is on beta stage. So, it's included with not all elements in template.

To get it works template should be uploaded on server that supports PHP 5+.

  1. Open builder.html
  2. Check checkbox if you want to use boxed mode
  3. Select layout that you need
  4. When you hover on header or content you will see handler, press on it.
  5. Header elements can be included and removed
  6. Content element also can be removed. We recommend you to start from grid. So after you pressed handler on content, choose grid12.
  7. Once you have chosen grid you can choose other elements, for example panels,tabs,forms etc.
  8. After you finished, click on cog(builder settings), and press Export.

How to use refresh panel function?

All you need is call function panel_refresh(panel,event,callback). For example: 

                
    panel_refresh($("#panel_to_refresh"), "shown", call_back_function); // Call on Refresh is Shown
    panel_refresh($("#panel_to_refresh"), "hidden", call_back_function); // Call on Refresh is Hidden

When function called first time it's set to block class .panel-refreshing. When you call function second time for same block, state will be changed to default.

More information and sample you can find on ui-panels.html page.

How to use remove panel function?

All you need is call function panel_remove(panel,event,callback). For example: 


    panel_remove($("#panel_to_remove"), "after", call_back_function);   // Call after panel removed
    panel_remove($("#panel_to_remove"), "before", call_back_function);  // Call before panel removed

More information and sample you can find on ui-panels.html page.

How to use collapse panel function?

All you need is call function panel_collapse(panel,event,callback). For example: 


    panel_collapse($("#panel_to_collapse"), "shown", call_back_function);   // Call on panel shown
    panel_collapse($("#panel_to_collapse"), "hidden", call_back_function);  // Call on panel hidden

More information and sample you can find on ui-panels.html page.

How to use custom progress bar?

There two actions for progress bar show - create new progressbar with options, update - update progressbar with options, destroy - remove.

    $.mpb(action,options); // call functions
    
    $.mpb('show',{value: [0,10]}); // Call default progress bar with 10% progress
    $.mpb('update',{value: [10,100]}); // Update progress bar to 100% progress
    $.mpb('destroy'); // Remove progress bar

Options:

value: array/integer. If isset array first element of it should be integer value from, second to. If isset integer it will be value to

position: NULL/string. Available position: bottom. Default is NULL(top).

speed: integer. Set speed of animation. Default: 20;

state: NULL/string. There 4 available custom states: success,warning,info,danger. Default NULL - base theme color.

complete: NULL/function. Callback function calling after animation finished.

 

You can find this plugin in js/plugins.js

More information and sample you can find on ui-alerts-popups.html

How to get boxed layout?

Add class page-container-boxed to body tag to get layout boxed.

How to get navigation toggled?

Add class page-navigation-toggled to page-container to get navigation toggled on load.

How to get navigation direction right?

Add class page-mode-rtl to page-container to get navigation aligned right.

How to get top navigation fixed?

Add class page-navigation-top-fixed to page-container to get top navigation fixed.

How to get custom navigation style?

Add class x-navigation-custom to x-navigation to get custom styled navigation.
